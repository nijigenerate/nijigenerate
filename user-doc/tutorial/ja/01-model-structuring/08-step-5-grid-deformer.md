# 08. ステップ 5：GridDeformer を必要な場所だけに入れる

GridDeformer は、顔や胴体のような広い面をまとめて変形するために使います。  
便利ですが、入れすぎると管理が難しくなるため、必要な場所に絞って配置します。

## 入れる場所の目安

基本的には、次の場所に GridDeformer を入れます。

- 顔
- 後頭部
- 横髪
- 胴体
- 前髪
- 胸部が別パーツの場合の胸まわり

顔や胴体のような広い面は、Part ひとつひとつにメッシュを張るだけでは扱いづらくなります。先に GridDeformer で変形用のまとまりを作っておくと、表情や体の向きを後から自然に調整しやすくなります。

## 挿入方法

すでに同じ位置にまとめ用の `Node` が入っていて、それをグリッドにしたい場合は、その `Node` を `GridDeformer` に変換します。まとめ用の `Node` がない場合は、変形させたい Part を選択し、その親として GridDeformer を挿入します。

GridDeformer 自体には通常の AutoMesh は掛けません。広い面を動かす予定がある場合は、Part に AutoMesh を掛ける前に、その外側に GridDeformer を用意しておきます。

## 格子数とマージン

格子数は、顔や後頭部では 8×8 程度、胴体では 10×14 程度を目安にします。キャラクターの縦横比やディテールに合わせて調整してください。

Grid のマージンは、全体サイズの約 10% 前後を目安にします。余白があると、変形時にエッジが自然に動きやすくなります。

## 目・口の DynamicComposite とグリッド

目や口など、顔の差分管理に `DynamicComposite` を使う場合も、グリッドにしておくと変形の影響が見えやすくなります。まずは 6×6 から 8×8 程度を目安にして、必要に応じて増減します。

## 操作例

1. 顔の Part を選択する。
2. 「親として GridDeformer 挿入」を選び、`Face::G` などの名前を付ける。
3. `Face::G` の子に `Face` Part が残っていることを確認する。
4. 後頭部、横髪、胴体、前髪、胸部にも、必要に応じて GridDeformer を挿入する。
