# 11. ステップ 8：左右反転ペアを登録する

ツリー構成、GridDeformer、AutoMesh の設定が済んだら、最後に左右反転ペアを登録します。  
左右の Part や Deformer の対応関係が固まってから登録することで、後から構成を変えたときの登録し直しを減らせます。

## 登録する理由

左右反転ペアを登録しておくと、パラメータ編集中に「反対方向からコピー」→「水平方向」を選んだとき、対応するペアの変形を参照して、左右反転した値を設定できます。

顔の左右向き、腕、脚、横髪、袖、アクセサリなど、左右で対になる要素はここで登録します。

## 登録しやすい名前の例

- `Eye::L` / `Eye::R`
- `Arm::L` / `Arm::R`
- `Leg::L` / `Leg::R`
- `HairSide_l` / `HairSide_r`
- `Sleeve_L` / `Sleeve_R`

`::L` / `::R`、`_l` / `_r`、`_L` / `_R` のように対応が分かる名前にしておくと、パターンマッチで登録しやすくなります。

## 操作例

1. 左右で対になる Part / Deformer の名前が、規則的に対応していることを確認する。
2. エディタ上で「左右反転ペアの設定」を開く。
3. パターンマッチで左右ペアを登録する。
4. 主要な左右ペアが登録されているか確認する。
5. 顔や胴体のパラメータ編集時に、「反対方向からコピー」→「水平方向」で意図した相手側へ反転コピーできるか確認する。
