# 10. ステップ 7：AutoMesh の設定と一括操作

ツリー構成と GridDeformer の配置ができたら、メッシュを張ります。  
ここでは、広い面を動かすためのグリッドと、各 Part の形に沿ったメッシュを分けて扱います。

## 基本ルール

`Part` には、基本的に `Optimum` アルゴリズムで AutoMesh を掛けます。  
`GridDeformer` には、通常の Optimum メッシュではなく、Grid 用 AutoMesh、つまり Grid プロセッサを使って Grid として AutoMesh を掛けます。

## 仕上げの流れ

1. 顔、胴体、前髪など、まとめて変形したい範囲に GridDeformer が入っていることを確認する。
2. GridDeformer の子に、変形させたい Part 群が入っていることを確認する。
3. GridDeformer には Grid 用 AutoMesh を掛ける。
4. 各 Part には Optimum を掛ける。

この順番にすると、広い面を動かすためのグリッドと、各パーツの形に沿ったメッシュを分けて管理できます。

## よくある失敗

- GridDeformer に通常の Optimum メッシュを掛けてしまう
- Part に Grid 用の設定を使ってしまう
- 顔や胴体などの広い面に GridDeformer を入れ忘れる
- 格子状に動かしたい範囲と、パーツ形状に沿わせたい範囲を同じ AutoMesh 設定で処理してしまう

## 複数パーツの一括操作

同じようなパーツに同じ AutoMesh 設定を掛けたいときは、複数選択による一括操作が便利です。  
AutoMesh だけでなく、ノードの挿入、追加、複数の子ノードを同じ親ノードに移す場合にも使えます。

まとめて操作したあとは、意図した対象だけが変わっているかをツリー上で確認してください。
