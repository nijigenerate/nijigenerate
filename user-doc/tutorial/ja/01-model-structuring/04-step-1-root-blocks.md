# 04. ステップ 1：Root 直下に大ブロックを作る

最初に、モデルを大きなまとまりに分けます。  
ここで胴体、手足、翼、尻尾、背後リボン、アクセサリなどの大枠を作っておくと、後から「胴体ごと動かす」「頭だけ角度を変える」といった操作がしやすくなります。

## 考え方

胴体や手足のような大きな単位は、Root の直下で整理します。まとめたい Part 群を先に選択し、「親として Node 挿入」で `Body::Root`、`Arm R::Root`、`Arm L::Root`、`Leg R::Root`、`Leg L::Root` などのまとまりを作ります。

この段階で重要なのは、最終的な親子関係を完全に決めることではなく、「あとで整理しやすいまとまり」を作ることです。頭まわりは次のステップで首の子として整理し、翼や尻尾などは後のステップで胴体や衣装の土台 Part の子になるように付け替えます。

Node は、挿入時に選択していた Part の原点位置を引き継ぎます。そのため、あとから回転や移動の基準にしやすくなります。

## よくある失敗

- すべての Part を Root のすぐ下に並べてしまう
- まとまりを作らず、後から毎回複数 Part を選択して動かそうとする
- 頭、腕、脚、衣装、アクセサリの大枠が混在している

Root 直下にすべてを並べると、後で頭だけ、胴体だけ、腕だけを動かしたいときに破綻しやすくなります。

## 操作例

1. 胴体用の Part 群を選択し、「親として Node 挿入」で `Body::Root` を作る。
2. 腕や脚も同様に、左右それぞれの Root ノードを作る。
3. 翼、尻尾、背後リボン、アクセサリなど、独立して動かす可能性があるものも大枠としてまとめる。
4. 頭まわりは、この段階では仮のまとまりでもよい。次のステップで首・顔の親子関係を整える。
