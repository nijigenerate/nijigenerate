# 05-1. 胴体の Yaw

この節では、`Body::Yaw-Pitch` の左右方向、つまり `Yaw=-1..1` を設定します。
Yaw は胴体を左右へ傾ける操作ではなく、体の向きを左右へ振り向ける操作です。

## 基準を作る

まず `(0,0)` に正面状態を登録します。次に `(-1,0)` または `(1,0)` の片側で、胴体が横を向く形を作ります。

左右対称に近いモデルでは、片側を設定したあと、顔と同じように「反対方向からの反転コピー」→「水平方向」を使って反対側を作ります。反転コピー後は、服の合わせ、アクセサリ、髪や腕との重なりを確認し、必要な箇所だけ補正します。

## 横向きの作り方

横向きの目安は、正面から 30 度程度です。胸の中央線、服の合わせ、肩幅、腕の付け根を見て、正面のまま平行移動しただけに見えないか確認します。胴体中央が少し回転して見えるように、手前側の胸や服の中央は大きめに動かし、奥側の脇や肩は詰め過ぎないようにします。奥側を細くし過ぎると、胴体が薄い板のように見えます。

Yaw の左右方向では、胴体を傾けて横向きに見せようとしないでください。肩の高さや腰の高さを大きく斜めに変えるのではなく、胸、腹、腰、服の側面が同じ体幹の厚みを保ったまま向きを変えるようにします。たとえば胸の正面中央は手前側へ寄り、奥側の脇や服の側面は少し隠れますが、腰やスカートまで別方向へ回って見えると不自然になります。

## 服とスカートの厚み

スカートや厚みのある服がある場合は、その膨らみも胴体の厚みとして扱います。横を向いたとき、手前側の布は少し広く見え、奥側の布は隠れたり圧縮されたりします。

ただし、スカートだけが胴体と別の角度で回って見えると、上半身と下半身が分離して見えます。腰、腹、スカートの上端が同じ体幹に属しているように、全体の流れを揃えてから布のふくらみを補正します。

横を向いたときに初めて見える部品は、全体の変形を作ったあとで個別に調整します。服の両サイドの縦線、ポケット、ジャケット、フード、飾り紐、アクセサリなどは、この対象になることがあります。アクセサリは、形そのものを変形するよりも、Root ノードの位置や角度を調整した方が扱いやすい場合があります。

## 中間キー

端点を作ったら、`(-0.5,0)` と `(0.5,0)` を追加します。中間キーでは、肩、胸、服の側面、腕の付け根が急に切り替わって見えないように整えます。

脚は接地感に影響しやすいため、胴体の向きに強く巻き込まないようにします。首、腕の付け根、襟、胸まわりは、胴体に連動する範囲を確認しながら調整してください。
