# 04. 顔の向き

この節では、顔を左右・上下へ向けるための `Face::Yaw-Pitch` と、首をかしげるための `Face::Roll` を設定します。  
顔の向きはモデル全体の印象に強く影響するため、最初から大きく動かそうとせず、元絵の構造に合わせて自然に見える範囲を決めます。

## 角度の目安

正面向きのイラストから作る場合、左右の振り向きは 30 度程度、上下の振り向きは 15 から 20 度程度を目安にします。絵柄によっては左右 45 度程度まで作れることもありますが、鼻、目、輪郭、髪の奥行きが不足していると破綻しやすくなります。

それ以上の大きな振り向きを作る場合は、横向き用の鼻、奥側の目を隠すためのパーツ、横顔に近い輪郭、髪の差分などを別途用意します。これらの準備がない場合は、無理に大きな角度を設定せず、正面絵から自然に補間できる範囲に留めてください。

## 作成するパラメータ

- `Face::Yaw-Pitch`
  `min=(-1,-1)`, `default=(0,0)`, `max=(1,1)`
- `Face::Roll`
  `min=-1`, `default=0`, `max=1`

`Face::Yaw-Pitch` は 2 次元パラメータです。X 方向で左右の振り向き、Y 方向で上下の振り向きを扱います。  
`Face::Roll` は 1 次元パラメータです。首を左右にかしげる動きだけを扱い、左右の振り向きとは分けて設定します。

## 編集方針

顔全体の大きな変形は、`Face::G` のような顔用 GridDeformer や MeshGroup で作ります。GridDeformer や MeshGroup で大まかな向きを作り、それだけでは整わない目、口、鼻、輪郭、前髪などを個別に補正します。

顔を回転して見せるには、前面と後面で変形量を変えることが重要です。前面にある鼻、口元、前髪は、奥行きが出るように位置と形を調整します。顔に乗っている目、眉、口、耳は、顔面から滑って見えないように追従させます。

GridDeformer や格子状の MeshGroup で顔全体の向きを作るときは、顔面を奥行きのあるカーブとして扱うと調整しやすくなります。詳しくは [03. 基本的な設定手順 - 深度を意識して大きな変形を作る](./03-basic-editing-flow.md#深度を意識して大きな変形を作る) を参照してください。

GridDeformer だけで目や口を変形すると、形がゆがみやすくなります。顔全体の向きを作ったあと、目や口の形は個別に確認し、必要なら Part や DynamicComposite 側で補正します。

## `Face::Yaw-Pitch` の設定

まず `(0,0)` に正面状態を登録します。ここがすべての基準になるため、目、口、鼻、輪郭、髪が正面状態として自然に見えることを確認してください。

次に、左右のどちらか片側を設定します。`(-1,0)` または `(1,0)` で顔全体が横を向く形を作ります。左右対称に描かれているモデルでは、片側を丁寧に設定したあと、メニューから「反対方向からの反転コピー」→「水平方向」を選び、反対側を作ると効率的です。

上下方向は、左右反転では作れないため個別に設定します。`(0,-1)` と `(0,1)` で、下向きと上向きの顔をそれぞれ作ります。上下の編集後に「デフォーマの左右対称化」を使うと、左右の崩れを抑えやすくなります。

端点を設定したら、`(-0.5,0)`, `(0.5,0)`, `(0,-0.5)`, `(0,0.5)` の中間キーを追加します。中間キーでは、輪郭、目の位置、口元、前髪、首元が急に動かないように整えます。

最後に、四隅の `(-1,-1)`, `(-1,1)`, `(1,-1)`, `(1,1)` を確認します。左右と上下を同時に入れると、軸上では自然だった変形が崩れることがあります。四隅は、軸上の設定をそのまま足し合わせる場所ではなく、最終的な見た目を補正する場所として扱います。

## `Face::Roll` の設定

`Face::Roll` では、首の付け根を中心に顔を左右へ傾けます。まず `0` に正面状態を登録し、`-1` または `1` の片側で傾きを作ります。左右対称にできる場合は、反対側を反転コピーで作ります。

端点だけでは傾きの途中で輪郭や首元が硬く見えることがあるため、`-0.5` と `0.5` にも中間キーを置きます。中間キーでは、首、髪、耳、顔の輪郭が自然につながるかを確認します。

## 確認すること

- 顔の左右方向が、元絵から自然に作れる範囲に収まっているか
- 大きく横を向かせるために必要な鼻、奥側の目、輪郭、髪の差分が不足していないか
- 目、眉、口、耳が顔面から滑って見えていないか
- 上下方向を設定したあと、顔の左右対称性が崩れていないか
- 四隅のキーで、左右と上下を同時に入れたときの破綻を補正しているか
