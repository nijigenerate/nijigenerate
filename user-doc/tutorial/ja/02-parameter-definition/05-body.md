# 05. 胴体の向き

この節では、胴体を左右・上下へ向けるための `Body::Yaw-Pitch` と、胴体を傾けるための `Body::Roll` を設定します。
胴体は首、腕、服、胸部、腰、スカート、アクセサリなど多くの要素を支えるため、顔よりも周辺パーツとの整合性を意識して編集します。

## 角度の目安

胴体の左右方向は、30 度程度を基準にします。この範囲であれば、服、腕の付け根、首まわりを大きく破綻させずに調整しやすくなります。
それ以上に横を向かせる場合は、服の側面、胸部、脇、アクセサリなど、正面では見えなかった部分を個別に作る必要があります。

## 作成するパラメータ

- `Body::Yaw-Pitch`
  `min=(-1,-1)`, `default=(0,0)`, `max=(1,1)`
- `Body::Roll`
  `min=-1`, `default=0`, `max=1`

`Body::Yaw-Pitch` は 2 次元パラメータです。X 方向で左右の向き、Y 方向で前傾・後傾を扱います。
`Body::Roll` は 1 次元パラメータです。胴体全体を左右へ傾ける動きだけを扱います。

## 編集方針

最初に GridDeformer や MeshGroup で胴体全体の形を作り、そのあとに細部を調整します。胴体全体の向きが決まる前に細部を作り込むと、後で全体を動かしたときに調整し直しが増えます。

GridDeformer や格子状の MeshGroup で胴体を左右・上下へ向けるときは、胴体を奥行きのあるカーブとして扱うと、肩、胸、腰まわりの移動量を整理しやすくなります。詳しくは [03. 基本的な設定手順 - 深度を意識して大きな変形を作る](./03-basic-editing-flow.md#深度を意識して大きな変形を作る) を参照してください。

胴体の GridDeformer では、配下にある胸、首、肩、服、腕の付け根、腰、スカートなどを見て、簡単な深度モデルを先に決めます。多くの正面モデルでは、胸や服の正面中央が手前、首は胸より奥、肩は胸より少し奥、脇や服の側面はさらに奥として扱えます。スカートがある場合は、布がどの方向に膨らんでいるかも深度として考えます。

ここで扱う `Body::Yaw-Pitch` の Yaw は、胴体を左右へ傾ける動きではありません。胴体が左右へ振り向く、つまり胸、腹、腰、服の側面が向きを変える動きです。左右に傾ける動きは `Body::Roll` で扱います。

胴体は、胸だけ、腹だけ、腰だけを個別に回して作ると破綻します。上半身、下半身、スカートが別々の回転に見えないように、最初に体幹全体の向きを決め、その上で胸部、服、スカート、アクセサリを補正します。
