# 05. 胴体の向き

この節では、胴体を左右・上下へ向けるための `Body::Yaw-Pitch` と、胴体を傾けるための `Body::Roll` を設定します。  
胴体は首、腕、服、胸部、アクセサリなど多くの要素を支えるため、顔よりも周辺パーツとの整合性を意識して編集します。

## 角度の目安

胴体の左右方向は、30 度程度を基準にします。この範囲であれば、服、腕の付け根、首まわりを大きく破綻させずに調整しやすくなります。  
それ以上に横を向かせる場合は、服の側面、胸部、脇、アクセサリなど、正面では見えなかった部分を個別に作る必要があります。

## 作成するパラメータ

- `Body::Yaw-Pitch`
  `min=(-1,-1)`, `default=(0,0)`, `max=(1,1)`
- `Body::Roll`
  `min=-1`, `default=0`, `max=1`

`Body::Yaw-Pitch` は 2 次元パラメータです。X 方向で左右の向き、Y 方向で前傾・後傾を扱います。  
`Body::Roll` は 1 次元パラメータです。胴体全体を左右へ傾ける動きだけを扱います。

## 編集方針

最初に GridDeformer や MeshGroup で胴体全体の形を作り、そのあとに細部を調整します。胴体全体の向きが決まる前に細部を作り込むと、後で全体を動かしたときに調整し直しが増えます。

胸部が胴体とは別パーツになっているモデルでは、先に胴体全体を変形し、その後で胸部を調整します。胸部だけを先に動かすのではなく、胴体の向きに合わせて胸部の奥行きや重なりを整えると、立体感を出しやすくなります。

横を向いたときに初めて見える部品は、全体の変形を作ったあとで個別に調整します。服の両サイドの縦線、ポケット、ジャケット、フード、飾り紐、アクセサリなどは、この対象になることがあります。アクセサリは、形そのものを変形するよりも、Root ノードの位置や角度を調整した方が扱いやすい場合があります。

脚は接地感に影響しやすいため、胴体の向きに強く巻き込まないようにします。首、腕の付け根、襟、胸まわりは、胴体に連動する範囲を確認しながら調整してください。

## `Body::Yaw-Pitch` の左右方向

まず `(0,0)` に正面状態を登録します。次に `(-1,0)` または `(1,0)` の片側で、胴体が横を向く形を作ります。

左右対称に近いモデルでは、片側を設定したあと、顔と同じように「反対方向からの反転コピー」→「水平方向」を使って反対側を作ります。反転コピー後は、服の合わせ、アクセサリ、髪や腕との重なりを確認し、必要な箇所だけ補正します。

端点を作ったら、`(-0.5,0)` と `(0.5,0)` を追加します。中間キーでは、肩、胸、服の側面、腕の付け根が急に切り替わって見えないように整えます。

## `Body::Yaw-Pitch` の前傾・後傾

正面位置、つまり `Yaw=0` では、前傾と後傾を分けて作ります。

前傾では、肩が手前に出て、体が前かがみに見えるようにします。首が見えづらくなるため、頭の位置も合わせて調整します。顔を 3% から 7% 程度拡大すると、頭が手前に来ている印象を作りやすくなります。

後傾では、前傾ほど効果が見えにくくなります。肩幅を少し狭くし、上半身の上側を低くするように GridDeformer を調整します。顔を 3% から 5% 程度小さくすると、頭が奥へ下がった印象を作りやすくなります。

左右向きの位置、つまり `Yaw=-1` と `Yaw=1` では、前傾・後傾を個別に確認します。前傾では体が自然に斜め前を向く形にし、後傾では体が後ろに反っているように調整します。片側の左右向きで調整し、もう片側を反転コピーすると効率的です。

## `Body::Roll` の設定

`Body::Roll` では、胴体全体を左右へ傾けます。まず `0` に正面状態を登録し、`-1` または `1` の片側で傾きを作ります。左右対称にできる場合は、反対側を反転コピーで作ります。

端点を設定したあと、`-0.5` と `0.5` に中間キーを置きます。中間キーでは、肩、胸、服、首元のつながりを確認します。顔と胴体を同時に動かし、首まわりや腕の付け根が不自然にずれていないかも確認してください。

## 確認すること

- 胴体の左右方向が、服や腕の付け根を破綻させない範囲に収まっているか
- 胸部が別パーツの場合、胴体全体の向きに合わせて胸部が自然に調整されているか
- 横向きで見える服の側面、ポケット、飾り紐、アクセサリなどを必要に応じて補正しているか
- 前傾・後傾で、頭の位置と大きさが胴体の動きに合っているか
- 顔と胴体を同時に動かしたとき、首や腕の付け根が破綻していないか
