# 02. 値域とキー位置

パラメータを作る前に、値域・デフォルト値・キー位置を決めます。  
この章の目的は、「どの値が何を意味するか」を固定してから変形設定へ進むことです。

## 基本方針

多くのパラメータは `-1` から `1`、デフォルト `0` で作ります。  
ただし、目や口の開閉のように負の値が不要なものは、開閉軸だけ `0` から `1` にします。

- 左右・上下・傾きのように中心から両方向へ動くもの  
  → `min=-1`, `default=0`, `max=1`
- 目や口の開閉のように「開く / 閉じる」の量を持つもの  
  → 開閉軸は `min=0`, `default=0`, `max=1`
- 喜び・中立・怒りなど表情方向を持つもの  
  → 表情軸は `min=-1`, `default=0`, `max=1`

## 推奨値域

| パラメータ | 次元 | X の意味と値 | Y の意味と値 |
| --- | --- | --- | --- |
| `Face::Yaw-Pitch` | 2D | `-1=左向き`, `0=正面`, `1=右向き` | `-1=下向き`, `0=正面`, `1=上向き` |
| `Face::Roll` | 1D | `-1=左へ傾ける`, `0=正面`, `1=右へ傾ける` | - |
| `Body::Yaw-Pitch` | 2D | `-1=左向き`, `0=正面`, `1=右向き` | `-1=下向き`, `0=正面`, `1=上向き` |
| `Body::Roll` | 1D | `-1=左へ傾ける`, `0=正面`, `1=右へ傾ける` | - |
| `Eye::L::Blink` / `Eye::R::Blink` | 2D | `0=開く`, `1=閉じる` | `-1=喜び`, `0=中立`, `1=深い閉じ` |
| `Eye::L::X-Y` / `Eye::R::X-Y` | 2D | `-1=左`, `0=中央`, `1=右` | `-1=下`, `0=中央`, `1=上` |
| `Eyebrow::L` / `Eyebrow::R` | 2D | `-1=下`, `0=中央`, `1=上` | `-1=怒り`, `0=中立`, `1=悲しみ` |
| `Mouth::Open` | 2D | `0=閉じる`, `1=開く` | `-1=喜び`, `0=中立`, `1=怒り` |

左右や上下の正負は、既存モデルやプロジェクトの約束に合わせて逆にしても構いません。  
ただし、同じモデル内では必ず統一します。顔だけ `-1=左`、目だけ `-1=右` のように混在させると、あとで組み合わせ確認ができなくなります。

## 推奨分割数

値域を決めたら、どの位置にキーを置くかを決めます。  
すべての軸を同じ細かさにすると作業量が増えすぎるため、動きの品質に効く軸だけ中間ポイントを増やします。

| 対象 | 推奨するキー位置 | 理由 |
| --- | --- | --- |
| `Face::Yaw-Pitch` | X/Y とも `-1`, `-0.5`, `0`, `0.5`, `1` | 顔の振り向きは補間の破綻が目立つため、中間の形を作る |
| `Face::Roll` | `-1`, `-0.5`, `0`, `0.5`, `1` | 傾きも中間で輪郭や首元が崩れやすい |
| `Body::Yaw-Pitch` | X/Y とも `-1`, `-0.5`, `0`, `0.5`, `1` | 胴体・首・腕の付け根の補間を安定させる |
| `Body::Roll` | `-1`, `-0.5`, `0`, `0.5`, `1` | 服や肩まわりの中間形を調整しやすくする |
| `Eye::{L,R}::Blink` の X | `0`, `0.25`, `0.5`, `0.75`, `1` | 開き目から閉じ目までの形状変化が大きい |
| 表情軸 | `-1`, `0`, `1` または `0`, `0.5`, `1` | 表情方向は段階が多すぎると管理しづらい |

表情軸とは、`Blink` の Y、`Eyebrow` の Y、`Mouth::Open` の Y のような、喜び・中立・怒りなどを表す軸です。  
ここは振り向きや瞬きほど細かく分割せず、代表的な表情だけを先に作ります。

## 2D パラメータのキー位置

2D パラメータでは、中央と軸上のキーを先に作ります。  
斜めキーは、左右と上下を同時に入れたときの補正として扱います。

- `(0, 0)`  
  基準値。正面、通常表情、初期状態。
- `(-1, 0)` / `(1, 0)`  
  X 方向だけの端。左右、開閉、上下位置など、X の意味だけを確認する。
- `(0, -1)` / `(0, 1)`  
  Y 方向だけの端。上下、表情方向など、Y の意味だけを確認する。
- `(-1, -1)` / `(-1, 1)` / `(1, -1)` / `(1, 1)`  
  組み合わせ補正。軸上の設定を入れたあと、破綻する場所だけ補正する。

`Face::Yaw-Pitch` と `Body::Yaw-Pitch` では、端点だけでなく `-0.5` と `0.5` の中間キーも作ります。  
まず `(0,0)`, `(-1,0)`, `(1,0)`, `(0,-1)`, `(0,1)` を作り、その後 `(-0.5,0)`, `(0.5,0)`, `(0,-0.5)`, `(0,0.5)` を追加して補間を整えます。

`Eye::{L,R}::Blink` や `Mouth::Open` のように X が `0..1` の場合は、X の負側を使いません。  
`Blink` では `(0,0)`, `(0.25,0)`, `(0.5,0)`, `(0.75,0)`, `(1,0)` を作り、開き目から閉じ目までの形を段階的に整えます。  
表情方向は `(0,-1)`, `(0,1)`, `(1,-1)`, `(1,1)` のような代表点を中心に確認し、必要な場所だけ中間を足します。

## 1D パラメータのキー位置

1D パラメータでは、中央と両端を先に作ります。

- `0`  
  基準値。正面、傾きなし。
- `-1`  
  左側または反時計回り側の傾き。
- `1`  
  右側または時計回り側の傾き。

`Face::Roll` と `Body::Roll` は、向きではなく傾きだけを扱います。  
端点だけでは補間が硬くなりやすいため、`-0.5` と `0.5` にも中間キーを置きます。  
左右を向かせる変形を入れたい場合は、`Face::Yaw-Pitch` や `Body::Yaw-Pitch` に戻します。

## 作業順

パラメータごとに、次の順で進めます。

1. パラメータを作成する。
2. `min/default/max` を設定する。
3. 中央キーを登録する。
4. X 方向の端キーを設定する。
5. Y 方向の端キーを設定する。
6. 斜めや組み合わせキーを補正する。
7. 他のパラメータと同時に動かして確認する。
