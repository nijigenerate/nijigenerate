# 01. 全体像と推奨パラメータ

## このチュートリアルで作るもの

モデル構成ができたあとに、顔・胴体・目・眉・口を動かすための基本パラメータを作ります。  
この章では、次の状態を目指します。

- 顔と胴体の向きを、左右・上下・傾きで分けて制御できる
- 目・眉・口を、左右差と表情差分を含めて制御できる
- 髪や服などの物理系を、通常の表情パラメータと混ぜずに整理できる

## 1. パラメータは「何を動かすか」ではなく「何の意味を持つか」で作る

パラメータは、単に Part を動かすための入力ではありません。  
同じ `Face::Yaw-Pitch` でも、顔 Part、目、眉、口、髪の付け根など複数の要素が少しずつ追従します。

そのため、パラメータ名は次のように意味単位で揃えます。

- `Face::Yaw-Pitch`  
  顔の左右・上下の振り向き
- `Face::Roll`  
  首を左右にかしげる動き
- `Body::Yaw-Pitch`  
  胴体の左右・上下の振り向き
- `Body::Roll`  
  胴体を左右に傾ける動き
- `Eye::{L,R}::Blink`  
  左右それぞれの目の開閉と表情寄りの閉じ方
- `Eye::{L,R}::X-Y`  
  左右それぞれの目線・目パーツの相対位置
- `Eyebrow::{L,R}`  
  眉の上下位置と表情変化
- `Mouth::Open`  
  口の開閉と表情変化

## 2. 2 次元パラメータと 1 次元パラメータを使い分ける

向きや表情の多くは、2 つの意味を同時に持つため 2 次元で作ります。

- 左右と上下を同時に扱う  
  `Face::Yaw-Pitch`, `Body::Yaw-Pitch`, `Eye::{L,R}::X-Y`
- 開閉と表情方向を同時に扱う  
  `Eye::{L,R}::Blink`, `Mouth::Open`
- 上下位置と表情方向を同時に扱う  
  `Eyebrow::{L,R}`

一方、傾きだけを表すものは 1 次元で十分です。

- `Face::Roll`
- `Body::Roll`

## 3. 先に決めること

パラメータを作る前に、次の 3 つを決めます。

- 名前  
  例：`Face::Yaw-Pitch`, `Eye::L::Blink`
- 値域とデフォルト値  
  例：`-1..1` / `0..1`, デフォルト `0`
- どのキーで何を登録するか  
  例：`(0,0)` は正面、`(1,0)` は右向き、`(1,1)` は組み合わせ補正

ここを決めずに変形設定へ進むと、同じパラメータの中で「右」「上」「笑い」「怒り」の意味が混ざりやすくなります。
