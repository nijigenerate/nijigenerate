# 06. 表情

この節では、目、眉、口の基本的な表情パラメータを設定します。  
顔や胴体の向きは空間的な向きを作るためのパラメータですが、表情は「どの形を見せるか」を整理するためのパラメータです。細かく分割しすぎず、まずは代表的な形を安定して作ることを優先します。

## 作成するパラメータ

- `Eye::L::Blink` / `Eye::R::Blink`
  `min=(0,-1)`, `default=(0,0)`, `max=(1,1)`
- `Eye::L::X-Y` / `Eye::R::X-Y`
  `min=(-1,-1)`, `default=(0,0)`, `max=(1,1)`
- `Eyebrow::L` / `Eyebrow::R`
  `min=(-1,-1)`, `default=(0,0)`, `max=(1,1)`
- `Mouth::Open`
  `min=(0,-1)`, `default=(0,0)`, `max=(1,1)`

左右の目と眉は別々のパラメータにします。片目だけ閉じる、片眉だけ動かす、といった操作ができるようにするためです。  
口は左右で分けず、`Mouth::Open` で開閉と表情方向をまとめて扱います。

## 目の瞬き

`Eye::L::Blink` と `Eye::R::Blink` は、X 方向で開閉量を扱います。`0` が開いた状態、`1` が閉じた状態です。Y 方向は表情寄りの閉じ方として使い、笑った閉じ目や深く閉じた形を分けます。

まず `(0,0)` に開き目を登録します。次に `(0.25,0)`, `(0.5,0)`, `(0.75,0)` で閉じ途中の形を作り、最後に `(1,0)` で普通の閉じ目を作ります。瞬きは形の変化が大きいため、途中のキーを省略すると補間中にまぶたやまつ毛が不自然に潰れやすくなります。

普通の閉じ目ができたら、`(1,-1)` で笑った閉じ目、`(1,1)` で深く閉じた形を作ります。笑い閉じと深い閉じが同じ形にならないよう、まぶたのカーブ、まつ毛、目尻の向きを確認してください。

## 目の位置

`Eye::L::X-Y` と `Eye::R::X-Y` は、目の相対位置を調整するためのパラメータです。`(0,0)` を中央位置として登録し、`(-1,0)`, `(1,0)`, `(0,-1)`, `(0,1)` に左右・上下の位置を設定します。

目線移動ではなく、顔の向きに合わせた目の位置補正として使う場合もあります。`Face::Yaw-Pitch` と組み合わせたとき、目が顔面から滑って見えないかを確認します。

## 眉

`Eyebrow::L` と `Eyebrow::R` は、X 方向で眉の上下位置、Y 方向で表情の方向を扱います。  
まず `(0,0)` に中立位置を登録し、`(-1,0)` と `(1,0)` で眉の高さを調整します。次に `(0,-1)` で怒り方向、`(0,1)` で悲しみ・困り方向の形を作ります。

眉は左右差が表情に強く出るため、左右を完全に同じにする必要はありません。ただし、片眉だけを動かしたときにもう片方へ影響しないよう、左右のパラメータは独立して設定します。

## 口

`Mouth::Open` は、X 方向で口の開閉、Y 方向で表情方向を扱います。`(0,0)` に中立の閉じ口を登録し、`(1,0)` に中立の開き口を作ります。

その後、`(0,-1)` と `(1,-1)` に喜び方向の閉じ口・開き口を作り、`(0,1)` と `(1,1)` に怒り方向の閉じ口・開き口を作ります。口は開閉量と表情方向が同時に変わるため、閉じ口だけでなく開き口でも表情が自然に見えるか確認してください。

## 確認すること

- 左右の目と眉が、それぞれ独立して動かせるか
- 瞬きの途中キーで、まぶたやまつ毛が不自然に潰れていないか
- 笑い閉じと深い閉じが、見た目として区別できるか
- 眉の上下位置と、怒り・悲しみなどの表情方向が混ざっていないか
- 口の開閉と表情方向を同時に動かしても破綻していないか
- `Face::Yaw-Pitch` と組み合わせたとき、目、眉、口が顔面から外れて見えていないか
