# 05-3. 胴体の Roll と確認

この節では、`Body::Roll` の設定と、胴体パラメータ全体の確認項目を整理します。

## `Body::Roll` の設定

`Body::Roll` では、胴体全体を左右へ傾けます。まず `0` に正面状態を登録し、`-1` または `1` の片側で傾きを作ります。左右対称にできる場合は、反対側を反転コピーで作ります。

`Body::Yaw-Pitch` の Yaw と混同しないようにしてください。Yaw は体の向きを左右へ振り向ける操作であり、Roll は体を左右へ傾ける操作です。Yaw で作るべき横向きを Roll のように傾けて作ると、体の厚みや服の側面が表現できません。

端点を設定したあと、`-0.5` と `0.5` に中間キーを置きます。中間キーでは、肩、胸、服、首元のつながりを確認します。顔と胴体を同時に動かし、首まわりや腕の付け根が不自然にずれていないかも確認してください。

## 確認すること

- 胴体の左右方向が、服や腕の付け根を破綻させない範囲に収まっているか
- Yaw が胴体の振り向きとして見えており、左右への傾きに見えていないか
- 横向きが正面から 30 度程度に見え、胸、腹、腰、服の側面が同じ体幹の厚みを保っているか
- 奥側の肩、脇、服の側面を詰め過ぎて、胴体が薄い板のように見えていないか
- 胸部が別パーツの場合、胴体全体の向きに合わせて胸部が自然に調整されているか
- 首が胸より手前に出て見えていないか
- スカートや厚みのある服を、胴体の深度として扱えているか
- 上半身、下半身、スカートが別々の回転に見えていないか
- 横向きで見える服の側面、ポケット、飾り紐、アクセサリなどを必要に応じて補正しているか
- 前傾・後傾で、頭の位置と大きさが胴体の動きに合っているか
- 前傾が単なる沈みではなく、腰を支点に上体が前下へ倒れた姿勢に見えているか
- 後傾が単なる持ち上げではなく、胸が開いて上体が後ろへ起きた姿勢に見えているか
- 横向きの前傾・後傾で、回転ツールに頼らず、各パーツの深度差として変形を作っているか
- 斜めキーで、横 30 度の向きと前傾・後傾の姿勢が同時に成立しているか
- 顔と胴体を同時に動かしたとき、首や腕の付け根が破綻していないか

設定後は、キー位置ごとに表示を確認します。特に斜めキーは、数値上は左右・上下の組み合わせになっていても、実際の見た目では横滑り、沈み込み、奥側の潰れとして出ることがあります。自然に見えない場合は、そのキー位置だけを見直し、腰、胸、肩、首、腕の付け根の関係が良くなるまで調整します。
