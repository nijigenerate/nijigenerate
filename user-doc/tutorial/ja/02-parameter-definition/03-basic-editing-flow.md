# 03. 基本的な設定手順

値域とキー位置を決めたら、各キー位置で実際の変形を設定します。ここでは、顔・胴体・表情のどれにも共通する基本操作を整理します。

パラメータ設定では、先に「どのパラメータを編集中にするか」と「どのキー位置を編集するか」を決めます。そのうえで、GridDeformer、MeshGroup、Part などを変形します。変形対象を先に触ってからパラメータを選ぶのではなく、パラメータとキー位置を決めてから変形を入れるのが基本です。

## 基本の流れ

パラメータごとに、次の順で進めます。

1. 編集するパラメータを選ぶ。
2. そのパラメータを編集中の状態にする。
3. 編集するキー位置を選ぶ。
4. そのキー位置で GridDeformer、MeshGroup、Part などを変形する。
5. 必要に応じて、別の Deformer や Part を追加で補正する。
6. 次のキー位置へ移動し、同じ手順で変形を設定する。
7. 軸上のキーを設定したあと、斜めや組み合わせキーを確認する。

「編集中の状態」とは、そのパラメータのキー位置を操作すると、対応する変形設定を編集できる状態のことです。パラメータを編集中にしていない場合、見た目を動かしているつもりでも、意図したパラメータのキーに変形が登録されないことがあります。

## キー位置を選んでから変形する

キー位置は、パラメータの値で選びます。たとえば `Face::Yaw-Pitch` なら、`(0,0)` が正面、`(-1,0)` と `(1,0)` が左右、`(0,-1)` と `(0,1)` が上下です。

各キー位置では、その値で見せたい形を作ります。`(-1,0)` を選んでいるときは左向きの顔を作り、`(0,1)` を選んでいるときは上向きの顔を作ります。キー位置を選ばずに Deformer だけを編集すると、どの値に対する変形なのかが曖昧になります。

中央の `(0,0)` は基準です。まず中央を登録し、そのあと端点や中間点を設定します。端点を先に作ってから中央を直すと、すでに設定した変形との関係が崩れやすくなります。

## 変形対象の使い分け

大きな形は GridDeformer や MeshGroup で作ります。顔全体、胴体全体、髪のまとまりなど、複数の Part をまとめて動かしたい場合に使います。

GridDeformer や MeshGroup だけでは整わない箇所は、Part を直接調整します。目、口、鼻、服の縁、アクセサリなどは、全体変形に乗せるだけでは形が崩れることがあります。その場合は、該当するキー位置で個別に補正します。

基本的な優先順は次の通りです。

1. GridDeformer や MeshGroup で大きな向きや位置を作る。
2. 形が崩れた Part を個別に補正する。
3. 前後関係や奥行きが必要な Part を追加で調整する。
4. 軸上のキーを確認し、最後に斜めや組み合わせキーを補正する。

## 深度を意識して大きな変形を作る

GridDeformer や、格子状に頂点を並べた MeshGroup を変形するときは、その配下に含まれる Part の形を「奥行きのある面」として考えると扱いやすくなります。正面絵そのものは 2D ですが、顔、胴体、髪、服のまとまりには、手前に出ている部分と奥へ下がっている部分があります。

`Face::Yaw-Pitch` や `Body::Yaw-Pitch` のような向きのパラメータでは、この奥行きを被写界深度情報のように見立てます。GridDeformer や格子状の MeshGroup が、その深度情報に沿ってゆるくカーブしていると考え、そのカーブ上を滑らせるように頂点を移動すると、左右・上下の回転を比較的少ない調整で作れます。

たとえば顔では、鼻、口元、前髪の一部を手前、耳や輪郭の奥側を後ろとして考えます。Yaw で横を向かせるときは、手前の部分を大きく移動し、奥の部分は小さく移動します。Pitch で上下を向かせるときも、額、鼻、あご、首元が同じ平面上に貼り付いて見えないように、深度の差に沿って変形量を変えます。

この考え方は、GridDeformer だけで完結させるためのものではありません。まずカーブに沿った大きな向きを作り、そのあと目、口、服の縁、アクセサリなど、深度だけでは説明しきれない部分を個別に補正します。

## 1 つのキー位置で完結させない

1 つのキー位置だけを見て自然に見えても、隣のキー位置との補間で崩れることがあります。`-1`、`0`、`1` の端点だけでなく、必要に応じて `-0.5` や `0.5` の中間キーも確認します。

特に顔や胴体の向きでは、端点を作ったあとに中間キーを見ます。中間で輪郭、首、服、髪、目や口の位置が急に動く場合は、中間キーに補正を入れます。

2D パラメータでは、軸上のキーだけで終わらせません。`(-1,-1)`, `(-1,1)`, `(1,-1)`, `(1,1)` のような斜めキーでは、左右と上下の変形が同時に入るため、軸上では見えなかった破綻が出ることがあります。
